Sismotos

Para rodar este projeto sigam os passos:
1. No MySQL crie o banco de dados dbmotos
2. Neste banco crie as tabelas existentes em bd/dbmotos.sql
3. Configure no arquivo /_config/connection.php o acesso de usuário e senha do banco
4. No arquivo /_helpers/index.php configure a URL de base

**As instruções acima são do professor Cleber para que o projeto pudesse rodar

Depois de seguir as instruções, com o projeto rodando, foi criada a imagem docker e o push foi feito e pode ser acessado em:

https://hub.docker.com/repository/docker/lilinoca/projeto-php/general
